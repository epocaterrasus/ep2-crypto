We need to assign PnL credit to each signal proportional to its contribution to the trade decision.\n\n**Approach: Weighted Signal Contribution**\n\nWhen your model combines signals into a final position, each signal's contribution to PnL is proportional to its weight times its directional agreement with the outcome.\n\n```python\nimport numpy as np\nimport pandas as pd\nfrom dataclasses import dataclass, field\nfrom typing import Dict, List\n\n@dataclass\nclass SignalAttribution:\n    \"\"\"\n    Decomposes PnL into contributions from each signal source.\n    \n    Method: Linear attribution based on signal weights and directional accuracy.\n    If the combined model is: position = w1*s1 + w2*s2 + ... + wn*sn,\n    then PnL_i = w_i * s_i * price_change / sum(w_j * s_j).\n    \"\"\"\n    signal_names: List[str]\n    \n    def attribute(self, signals_df: pd.DataFrame, weights: Dict[str, float],\n                  pnl_series: pd.Series) -> pd.DataFrame:\n        \"\"\"\n        signals_df: DataFrame with columns for each signal value at trade entry.\n        weights: dict mapping signal name to its model weight.\n        pnl_series: PnL for each trade (aligned index with signals_df).\n        \n        Returns DataFrame with PnL attributed to each signal.\n        \"\"\"\n        weighted_signals = pd.DataFrame(index=signals_df.index)\n        for name in self.signal_names:\n            weighted_signals[name] = signals_df[name] * weights[name]\n        \n        # Confidence Gating Attribution — Value of Abstaining\n\nThis measures how much PnL is saved (or lost) by the confidence gate refusing to trade.\n\n```python\n@dataclass\nclass ConfidenceGatingAttribution:\n    \"\"\"\n    Measures the value added by confidence-based trade filtering.\n    \n    Compares actual PnL (with gating) vs hypothetical PnL (without gating).\n    The difference is the \"gating alpha.\"\n    \"\"\"\n    \n    def analyze(self, all_signals_df: pd.DataFrame, \n                confidence_scores: pd.Series,\n                threshold: float,\n                hypothetical_pnl: pd.Series) -> Dict:\n        \"\"\"\n        all_signals_df: signals for ALL potential trades (taken and rejected).\n        confidence_scores: confidence score for each potential trade.\n        threshold: the confidence threshold used for gating.\n        hypothetical_pnl: what PnL WOULD have been for each potential trade.\n        \n        Returns dict with gating analysis.\n        \"\"\"\n        taken = confidence_scores >= threshold\n        rejected = ~taken\n        \n        pnl_taken = hypothetical_pnl[taken]\n        pnl_rejected = hypothetical_pnl[rejected]\n        \n        results = {\n            'total_opportunities': len(hypothetical_pnl),\n            'trades_taken': taken.sum(),\n            'trades_rejected': rejected.sum(),\n            'filter_rate': rejected.mean(),\n            'pnl_from_taken': pnl_taken.sum(),\n            'pnl_from_rejected_hypothetical': pnl_rejected.sum(),\n            'gating_alpha': -pnl_rejected[pnl_rejected < 0].sum(),  # losses avoided\n            'opportunity_cost': pnl_rejected[pnl_rejected > 0].sum(),  # gains missed\n            'net_gating_value': -pnl_rejected.sum(),  # net value of abstaining\n            'avg_pnl_taken': pnl_taken.mean(),\n            'avg_pnl_rejected': pnl_rejected.mean(),\n            'sharpe_with_gate': pnl_taken.mean() / pnl_taken.std() * np.sqrt(252) if pnl_taken.std() > 0 else 0,\n            'sharpe_without_gate': hypothetical_pnl.mean() / hypothetical_pnl.std() * np.sqrt(252) if hypothetical_pnl.std() > 0 else 0,\n        }\n        \n        # Optimal threshold search\n        thresholds = np.arange(0.1, 1.0, 0.05)\n        threshold_results = []\n        for t in thresholds:\n            mask = confidence_scores >= t\n            if mask.sum() < 10:\n                continue\n            subset_pnl = hypothetical_pnl[mask]\n            threshold_results.append({\n                'threshold': t,\n                'n_trades': mask.sum(),\n                'total_pnl': subset_pnl.sum(),\n                'sharpe': subset_pnl.mean() / subset_pnl.std() * np.sqrt(252) if subset_pnl.std() > 0 else 0,\n                'hit_rate': (subset_pnl > 0).mean(),\n            })\n        results['threshold_curve'] = pd.DataFrame(threshold_results)\n        \n        return results\n```\n\n**Key insight:** If `net_gating_value` is positive, confidence gating is adding value.        profiles[feat] = profile\n        \n        return profiles\n    \n    def big_winner_big_loser_analysis(self, trades_df: pd.DataFrame,\n                                       feature_cols: big_losers[pnl_col].sum() / trades_df[pnl_col].sum() * 100,\n            'feature_comparison': pd.DataFrame(comparison).T,\n        }\n```\n\n**What to look for in the output:**\n- Features with Drawdown Attribution\n\n```python\n@dataclass\nclass DrawdownAttribution:\n    \"\"\"\n    When drawdowns occur, diagnose which component failed.\n    Possible causes: model accuracy degradation, feature drift,\n    regime change, execution quality degradation.\n    \"\"\"\n    \n    def identify_drawdowns(self, equity_curve: pd.Series,\n                            min_drawdown_pct: float = 0.02) -> List[Dict]:\n        \"\"\"Find all drawdown periods exceeding threshold.\"\"\"\n        cummax = equity_curve.cummax()\n        drawdown = (equity_curve - cummax) / cummax\n        \n        drawdowns = []\n        in_drawdown = False\n        start_idx = None\n        peak_val = None\n        \n        for i in range(len(drawdown)):\n            if drawdown.iloc[i] < -min_drawdown_pct and not in_drawdown:\n                in_drawdown = True\n                start_idx = i - 1 if i > 0 else 0  # peak index\n                peak_val = cummax.iloc[i]\n            elif drawdown.iloc[i] >= 0 and in_drawdown:\n                in_drawdown = False\n                trough_idx = drawdown.iloc[start_idx:i].idxmin()\n                drawdowns.append({\n                    'start': equity_curve.index[start_idx],\n                    'trough': trough_idx,\n                    'end': equity_curve.index[i],\n                    'peak_value': peak_val,\n                    'trough_value': equity_curve.loc[trough_idx],\n                    'depth_pct': drawdown.loc[trough_idx] * 100,\n                    'duration': i - start_idx,\n                    'recovery_duration': i - equity_curve.index.get_loc(trough_idx),\n                })\n        \n        # Handle ongoing drawdown\n        if in_drawdown:\n            trough_idx = drawdown.iloc[start_idx:].idxmin()\n            drawdowns.append({\n                'start': equity_curve.index[start_idx],\n                'trough': trough_idx,\n                'end': None,  # still in drawdown\n                'peak_value': peak_val,\n                'trough_value': equity_curve.loc[trough_idx],\n                'depth_pct': drawdown.loc[trough_idx] * 100,\n                'duration': len(drawdown) - Model accuracy check\n        dd_hit_rate = (dd_trades[pnl_col] > 0).mean() if len(dd_trades) > 0 else 0\n        pre_hit_rate = (pre_trades[pnl_col] > 0).mean() if len(pre_trades) > 0 else 0\n        diagnosis['hit_rate_drawdown'] = dd_hit_rate\n        diagnosis['hit_rate_pre'] = pre_hit_rate\n        diagnosis['accuracy_degraded'] = dd_hit_rate < pre_hit_rate - 0.05\n        \n        # 2. Feature drift check\n        feature_drift = {}\n        for feat in feature_cols:\n            if feat in dd_trades.columns and feat in pre_trades.columns:\n                dd_mean = dd_trades[feat].mean()\n                pre_mean = pre_trades[feat].mean()\n                pre_std = pre_trades[feat].std()\n                drift_z = (dd_mean - pre_mean) / pre_std if pre_std > 0 else 0\n                feature_drift[feat] = {\n                    'pre_mean': pre_mean,\n                    'dd_mean': dd_mean,\n                    'drift_z_score': drift_z,\n                    'significant_drift': abs(drift_z) > 2.0,\n                }\n        diagnosis['feature_drift'] = pd.DataFrame(feature_drift).T\n        \n        # 3. Regime change check\n        if regime_col in trades_df.columns:\n            dd_regimes = dd_trades[regime_col].value_counts(normalize=True)\n            pre_regimes = pre_trades[regime_col].value_counts(normalize=True)\n            diagnosis['regime_distribution_drawdown'] = dd_regimes.to_dict()\n            diagnosis['regime_distribution_pre'] = pre_regimes.to_dict()\n            diagnosis['regime_changed'] = not dd_regimes.index[0] == pre_regimes.index[0] if len(dd_regimes) > 0 and len(pre_regimes) > 0 else False\n        \n        # 4. Confidence distribution check\n        if confidence_col in trades_df.columns:\n            diagnosis['avg_confidence_drawdown'] = dd_trades[confidence_col].mean()\n            diagnosis['avg_confidence_pre'] = pre_trades[confidence_col].mean()\n            diagnosis['confidence_dropped'] = (\n                dd_trades[confidence_col].mean() < pre_trades[confidence_col].mean() - 0.05\n            )\n        \n        # 5. Tail analysis — is the drawdown from many small losses or a few large ones?\n        if len(dd_trades) > 0:\n            dd_pnl = dd_trades[pnl_col]\n            losses = dd_pnl[dd_pnl < 0]\n            diagnosis['n_losing_trades'] = len(losses)\n            diagnosis['avg_loss_size'] = losses.mean() if len(losses) > 0 else 0\n            diagnosis['max_single_loss'] = losses.min() if len(losses) > 0 else 0\n            diagnosis['loss_pattern'] = (\n                'many_small_losses' if len(losses) > len(dd_trades) * 0.6 \n                and losses.std() < abs(losses.mean()) * 0.5\n                else 'few_large_losses' if len(losses) < len(dd_trades) * 0.4\n                else 'mixed'\n            )\n        \n        # Summary diagnosis\n        causes = []\n        if diagnosis.get('accuracy_degraded'):\n            causes.append('model_accuracy_degradation')\n        if any(v.get('significant_drift', False) for v in feature_drift.values()):\n            causes.append('feature_drift')\n        if diagnosis.get('regime_changed'):\n            causes.append('regime_change')\n        if diagnosis.get('confidence_dropped'):\n            causes.append('confidence_erosion')\n        diagnosis['probable_causes'] = causes if causes else ['unknown_or_bad_luck']\n        \n        return diagnosis\n```\n\n**Drawdown diagnosis flowchart:**\n1. Was hit rate materially lower? If yes, model accuracy degraded.\n2. Did features drift (z-score > 2)? If yes, the market microstructure changed.\n3. Did the regime change? If yes, the model may not be suited for the new regime.\n4. Was the loss pattern \"many small losses\" vs \"few large losses\"? Many small losses = systematic model failure. Few large losses = tail risk event, check stop-loss logic.\n\n---\n\n## 9. Execution Cost Attribution\n\n```python\n@dataclass\nclass ExecutionCostAttribution:\n    \"\"\"\n    Decompose execution costs into: fees, slippage, latency cost,\n    and market impact. Alpha vs Beta Decomposition\n\n```python\nfrom numpy.linalg import lstsq\n\n@dataclass\nclass AlphaBetaDecomposition:\n    \"\"\"\n    Decompose returns into alpha (skill) and beta (market exposure).\n    \n    For crypto: if BTC goes up 5% and your long-biased strategy makes 6%,\n    you have 1% alpha and 5% beta. The beta part is \"free\" — you could get \n    it by just holding BTC. Only alpha represents trading skill.\n    \"\"\"\n    \n    def decompose(self, strategy_returns: pd.Series,\n                   benchmark_returns: pd.Series,\n                   risk_free_rate: float = 0.0) -> Dict:\n        \"\"\"\n        Single-factor regression: R_strategy = alpha + beta * R_benchmark + epsilon.\n        \"\"\"\n        # Align indices\n        aligned = pd.DataFrame({\n            'strategy': strategy_returns,\n            'benchmark': benchmark_returns\n        }).dropna()\n        \n        y = aligned['strategy'].values - risk_free_rate\n        X = np.column_stack([np.ones(len(aligned)), aligned['benchmark'].values - risk_free_rate])\n        \n        # OLS regression\n        coeffs, residuals, _, _ = lstsq(X, y, rcond=None)\n        alpha_daily = coeffs[0]\n        beta = coeffs[1]\n        \n        # Predicted and residual\n        predicted = X @ coeffs\n        epsilon = y - predicted\n        \n        # Annualize (365 days for crypto)\n        alpha_annual = alpha_daily * 365\n        \n        # R-squared\n        ss_res = np.sum(epsilon**2)\n        ss_tot = np.sum((y - y.mean())**2)\n        r_squared = 1 - ss_res / ss_tot if ss_tot > 0 else 0\n        \n        # Information ratio (alpha / tracking error)\n        tracking_error = epsilon.std() * np.sqrt(365)\n        information_ratio = alpha_annual / tracking_error if tracking_error > 0 else 0\n        \n        # Return decomposition\n        total_return = aligned['strategy'].sum()\n        beta_return = beta * aligned['benchmark'].sum()\n        alpha_return = total_return - beta_return\n        \n        return {\n            'alpha_daily': alpha_daily,\n            'alpha_annual': alpha_annual,\n            'beta': beta,\n            'r_squared': r_squared,\n            'information_ratio': information_ratio,\n            'tracking_error': tracking_error,\n            'total_return': total_return,\n            'beta_return': beta_return,\n            'alpha_return': alpha_return,\n            'alpha_pct_of_total': alpha_return / total_return * 100 if total_return != 0 else 0,\n            'beta_pct_of_total': beta_return / total_return * 100 if total_return != 0 else 0,\n        }\n    \n    def multi_factor_decompose(self, strategy_returns: pd.Series,\n                                factor_returns: pd.DataFrame,\n                                risk_free_rate: float = 0.0) -> Dict:\n        \"\"\"\n        Multi-factor model: R = alpha + b1*F1 + b2*F2 + ... + epsilon\n        \n        Useful crypto factors:\n        - BTC return (market factor)\n        - ETH-BTC spread (alt-coin factor)  \n        - Funding rate (carry factor)\n        - Volatility change (vol factor)\n        - Dollar index DXY change (macro factor)\n        \"\"\"\n        aligned = pd.concat([strategy_returns, factor_returns], axis=1).dropna()\n        \n        y = aligned.iloc[:, 0].values - risk_free_rate\n        factor_vals = aligned.iloc[:, 1:].values\n        X = np.column_stack([np.ones(len(aligned)), factor_vals])\n        \n        coeffs, _, _, _ = lstsq(X, y, rcond=None)\n        alpha_daily = coeffs[0]\n        betas = dict(zip(factor_returns.columns, coeffs[1:]))\n        \n        predicted = X @ coeffs\n        epsilon = y - predicted\n        \n        ss_res = np.sum(epsilon**2)\n        ss_tot = np.sum((y - y.mean())**2)\n        r_squared = 1 - ss_res / ss_tot if ss_tot > 0 else 0\n        \n        # Each factor's contribution to total return\n        factor_contributions = {}\n        for i, name in enumerate(factor_returns.columns):\n            contribution = coeffs[i + 1] * aligned.iloc[:, i + 1].sum()\n            factor_contributions[name] = contribution\n        \n        return {\n            'alpha_daily': alpha_daily,\n            'alpha_annual': alpha_daily * 365,\n            'betas': betas,\n            'r_squared': r_squared,\n            'factor_contributions': factor_contributions,\n            'residual_vol': epsilon.std() * np.sqrt(365),\n            'unexplained_return': epsilon.sum(),\n        }\n    \n    def rolling_alpha_beta(self, strategy_returns: pd.Series,\n                            benchmark_returns: pd.Series,\n                            window: int = 60) -> pd.DataFrame:\n        \"\"\"\n        Rolling alpha and beta to see how market exposure changes over time.\n        Useful for detecting: your strategy becoming more directional,\n        alpha decaying while beta increases, etc.\n        \"\"\"\n        results = []\n        for i in range(window, len(strategy_returns)):\n            s = strategy_returns.iloc[i - window:i]\n            b = benchmark_returns.iloc[i - window:i]\n            \n            y = s.values\n            X = np.column_stack([np.ones(len(s)), b.values])\n            coeffs, _, _, _ = lstsq(X, y, rcond=None)\n            \n            results.append({\n                'date': strategy_returns.index[i],\n                'rolling_alpha': coeffs[0] * 365,\n                'rolling_beta': coeffs[1],\n            })\n        \n        return pd.DataFrame(results).set_index('date')\n```\n\n**Key insight:** If `r_squared` is high (> 0.5), most of your returns are explained by market movements (beta). You could get similar returns by just holding BTC with leverage. True trading skill shows up as high `alpha_annual` with low `r_squared` and high `information_ratio` (> 1.0 is good, > 2.0 is excellent).\n\n**What to watch in the rolling analysis:**\n- Alpha trending down = alpha decay, your edge is eroding.\n- Beta drifting away from zero = your \"market-neutral\" strategy is taking on directional exposure, possibly unintentionally.\n\n---\n\n## 11. Brinson-Fachler Attribution (Adapted for Trading)\n\nThe classic portfolio attribution framework, adapted from asset management to trading regimes.\n\n```python\n@dataclass\nclass BrinsonFachlerAttribution:\n    \"\"\"\n    Adapted Brinson-Fachler for a trading system.\n    \n    Original: decomposes excess return vs benchmark into allocation, \n    selection, and interaction trades within a regime\n    - \"Interaction\" = the combined effect\n    \n    Benchmark: a naive strategy (e.g., always-on, equal-weight signals)\n    \"\"\"\n    \n    def attribute(self, trades_df:        trades_df must have:\n        - regime: the regime label for each trade\n        - pnl: actual strategy PnL\n        - benchmark_pnl: what a naive strategy would have made\n        \n        The benchmark could be: always trading (no gating), \n        equal signal weights, or a simple momentum strategy.\n        \"\"\"\n        regimes = trades_df[regime_col].unique()\n        \n        # Portfolio (actual) weights and returns by regime\n        total_trades = len(trades_df)\n        \n        results = []\n        for regime in regimes:\n            mask = trades_df[regime_col] == regime\n            n_regime = mask.sum()\n            \n            # Weights: fraction of trades in this regime\n            w_portfolio = n_regime / total_trades\n            \n            # For benchmark, assume equal allocation across regimes\n            w_benchmark = 1.0 / len(regimes)\n            \n            # Returns: average PnL in this regime\n            r_portfolio = modified_pnl.cumsum().cummax()).min(),\n                'marginal_pnl': baseline_pnl.sum() - modified_pnl.sum(),\n                'marginal_sharpe': baseline_sharpe - modified_sharpe,\n            })\n        \n        return pd.DataFrame(results)\n    \n    def interaction_analysis(self, base_config: Dict[str, bool],\n                              data: pd.DataFrame) -> pd.DataFrame:\n        \"\"\"\n        Pairwise interaction: what is the combined marginal value of \n        removing two components together vs removing each separately?\n        \n        If marginal(A+B removed) > marginal(A removed) + marginal(B removed),\n        then A and B have positive interaction (they reinforce each other).\n        \"\"\"\n        components = [k for k, v in base_config.items() if v]\n        \n        baseline_pnl = self.strategy_runner(base_config, data)\n        baseline_total = baseline_pnl.sum()\n        \n        # Single removals\n        single_marginals = {}\n        for comp in components:\n            cfg = base_config.copy()\n            cfg[comp] = False\n            pnl = self.strategy_runner(cfg, data)\n            single_marginals[comp] = baseline_total - pnl.sum()\n        \n        # Pairwise removals\n        results = []\n        for i, a in enumerate(components):\n            for b in components[i + 1:]:\n                cfg = base_config.copy()\n                cfg[a] = False\n                cfg[b] = False\n                pnl = self.strategy_runner(cfg, data)\n                pair_marginal = baseline_total - pnl.sum()\n                \n                expected_if_independent = single_marginals[a] + single_marginals[b]\n                interaction = pair_marginal - expected_if_independent\n                \n                results.append({\n                    'component_a': a,\n                    'component_b': b,\n                    'marginal_a': single_marginals[a],\n                    'marginal_b': single_marginals[b],\n                    'marginal_pair': pair_marginal,\n                    'expected_independent': expected_if_independent,\n                    'interaction': interaction,\n                    'interaction_type': 'synergistic' if interaction > 0 \n                        else 'redundant' if interaction < 0 \n                        else 'independent',\n                })\n        \n        return pd.DataFrame(results)\n```\n\n**Example output for leave-one-out:**\n\n```\n                              configuration  total_pnl  sharpe  max_drawdown  marginal_pnl  marginal_sharpe\n0                   BASELINE (all enabled)    12450.0    2.31       -1240.0           0.0             0.00\n1               WITHOUT microstructure       8200.0    1.62       -1890.0        4250.0             0.69\n2                  WITHOUT nq_lead_lag      10100.0    1.95       -1520.0        2350.0             0.36\n3             WITHOUT regime_detection      11800.0    1.78       -2100.0         650.0             0.53\n4           WITHOUT confidence_gating       10900.0    1.45       -2400.0        1550.0             0.86\n5                WITHOUT macro_events       12100.0    2.25       -1300.0         350.0             0.06\n```\n\n**Interpretation:** Microstructure is the highest-value component by PnL ($4250 marginal).
