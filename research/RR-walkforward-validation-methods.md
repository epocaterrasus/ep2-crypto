Walk-Forward Sharpe Ratio Estimation\n\n### The Problem\n\nYou have K walk-forward folds, each producing a sequence of OOS returns. How to estimate the strategy's true Sharpe ratio?\n\n### Method 1: Concatenated Sharpe (RECOMMENDED)\n\n```python\ndef sharpe_concatenated(\n    fold_returns: list[np.ndarray],\n    annualization_factor: float = np.sqrt(288 * 365),\n) -> dict:\n    \"\"\"\n    Concatenate all OOS returns and compute a single Sharpe.\n    \n    This is the CORRECT method because:\n    1. Each return is truly out-of-sample\n    2. Concatenation preserves the time series structure\n    3. The resulting Sharpe has a clear statistical interpretation\n    \n    The annualization factor for 5-min bars:\n    sqrt(288 bars/day * 365 days/year) = sqrt(105,120) ~ 324.2\n    \"\"\"\n    all_returns = np.concatenate(fold_returns)\n    \n    mu = np.mean(all_returns)\n    sigma = np.std(all_returns, ddof=1)\n    \n    if sigma == 0:\n        return {\"sharpe\": 0.0, \"se\": np.inf, \"ci_lower\": 0.0, \"ci_upper\": 0.0}\n    \n    sharpe = (mu / sigma) * annualization_factor\n    \n    # Standard error of Sharpe ratio (Lo, 2002)\n    n = len(all_returns)\n    se_sharpe = np.sqrt(\n        (1 + 0.5 * (sharpe / annualization_factor) ** 2) / n\n    ) * annualization_factor\n    \n    return {\n        \"sharpe\": float(sharpe),\n        \"se\": float(se_sharpe),\n        \"ci_lower\": float(sharpe - 1.96 * se_sharpe),\n        \"ci_upper\": float(sharpe + 1.96 * se_sharpe),\n        \"n_observations\": n,\n        \"mean_return\": float(mu),\n        \"std_return\": float(sigma),\n    }\n```\n\n### Method 2: Average of Per-Fold Sharpes (PROBLEMATIC)\n\n```python\ndef sharpe_averaged(\n    fold_returns: list[np.ndarray],\n    annualization_factor: float = np.sqrt(288 * 365),\n) -> dict:\n    \"\"\"\n    Average the Sharpe ratio computed per fold.\n    \n    PROBLEMS with this method:\n    1. Sharpe ratio is NOT a linear statistic — averaging Sharpes\n       is not the same as the Sharpe of averaged returns\n    2. Short folds produce noisy Sharpe estimates with fat tails\n    3. A fold with near-zero volatility can produce extreme Sharpe\n    4. Gives equal weight to folds of different sizes\n    \n    Use this ONLY as a supplementary diagnostic, not as the primary metric.\n    \"\"\"\n    fold_sharpes = []\n    for returns in fold_returns:\n        if len(returns) < 10:\n            continue\n        mu = np.mean(returns)\n        sigma = np.std(returns, ddof=1)\n        if sigma < 1e-10:\n            continue\n        fold_sharpes.append((mu / sigma) * annualization_factor)\n    \n    if len(fold_sharpes) == 0:\n        return {\"sharpe_mean\": 0.0, \"sharpe_std\": np.inf}\n    \n    return {\n        \"sharpe_mean\": float(np.mean(fold_sharpes)),\n        \"sharpe_std\": float(np.std(fold_sharpes, ddof=1)),\n        \"sharpe_median\": float(np.median(fold_sharpes)),\n        \"n_folds\": len(fold_sharpes),\n        \"min_fold_sharpe\": float(np.min(fold_sharpes)),\n        \"max_fold_sharpe\": float(np.max(fold_sharpes)),\n    }\n```\n\n### Method 3: Probabilistic Sharpe Ratio (PSR) — De Prado\n\n```python\nfrom scipy.stats import norm\n\n\ndef probabilistic_sharpe_ratio(\n    fold_returns: list[np.ndarray],\n    benchmark_sharpe: float = 0.0,\n    annualization_factor: float = np.sqrt(288 * 365),\n) -> dict:\n    \"\"\"\n    Compute the Probabilistic Sharpe Ratio (PSR).\n    \n    PSR = probability that the true Sharpe exceeds the benchmark,\n    accounting for higher moments (skewness, kurtosis) of returns.\n    \n    From De Prado (2018), this is the gold standard for Sharpe evaluation.\n    \n    Formula:\n    PSR = Phi( (SR - SR*) * sqrt(n-1) / \n               sqrt(1 - skew*SR + (kurt-1)/4 * SR^2) )\n    \n    where SR* is benchmark Sharpe, n is number of observations.\n    \"\"\"\n    all_returns = np.concatenate(fold_returns)\n    n = len(all_returns)\n    \n    mu = np.mean(all_returns)\n    sigma = np.std(all_returns, ddof=1)\n    \n    if sigma == 0:\n        return {\"psr\": 0.0, \"sharpe\": 0.0}\n    \n    sr = mu / sigma  # non-annualized\n    sr_benchmark = benchmark_sharpe / annualization_factor  # de-annualize\n    \n    skew = float(pd.Series(all_returns).skew())\n    kurt = float(pd.Series(all_returns).kurtosis())  # excess kurtosis\n    \n    denominator = np.sqrt(\n        1 - skew * sr + ((kurt + 3) - 1) / 4 * sr ** 2\n    )\n    \n    if denominator == 0:\n        return {\"psr\": 0.0, \"sharpe\": float(sr * annualization_factor)}\n    \n    psr = float(norm.cdf(\n        (sr - sr_benchmark) * np.sqrt(n - 1) / denominator\n    ))\n    \n    return {\n        \"psr\": psr,\n        \"sharpe\": float(sr * annualization_factor),\n        \"skewness\": skew,\n        \"excess_kurtosis\": kurt,\n        \"n_observations\": n,\n        \"interpretation\": (\n            \"strong\" if psr > 0.95\n            else \"moderate\" if psr > 0.85\n            else \"weak\" if psr > 0.5\n            else \"no evidence\"\n        ),\n    }\n```\n\n### Variance Decomposition\n\n```python\ndef sharpe_variance_decomposition(\n    fold_returns: list[np.ndarray],\n    annualization_factor: float = np.sqrt(288 * 365),\n) -> dict:\n    \"\"\"\n    Decompose Sharpe variance into within-fold and between-fold components.\n    \n    High between-fold variance suggests regime dependency.\n    High within-fold variance suggests noisy signal.\n    \"\"\"\n    fold_sharpes = []\n    fold_sizes = []\n    \n    for returns in fold_returns:\n        if len(returns) < 10:\n            continue\n        mu = np.mean(returns)\n        sigma = np.std(returns, ddof=1)\n        if sigma < 1e-10:\n            continue\n        fold_sharpes.append((mu / sigma) * annualization_factor)\n        fold_sizes.append(len(returns))\n    \n    fold_sharpes = np.array(fold_sharpes)\n    fold_sizes = np.array(fold_sizes)\n    \n    # Between-fold variance (regime risk)\n    between_var = np.var(fold_sharpes, ddof=1)\n    \n    # Within-fold variance (estimation noise)\n    within_vars = []\n    for i, returns in enumerate(fold_returns):\n        if len(returns) < 10:\n            continue\n        n = len(returns)\n        sr = fold_sharpes[len(within_vars)] / annualization_factor\n        # Lo (2002) formula for SE of Sharpe\n        within_var = (1 + 0.5 * sr ** 2) / n * annualization_factor ** 2\n        within_vars.append(within_var)\n    \n    mean_within_var = np.mean(within_vars) if within_vars else 0\n    \n    total_var = between_var + mean_within_var\n    \n    return {\n        \"total_variance\": float(total_var),\n        \"between_fold_variance\": float(between_var),\n        \"within_fold_variance\": float(mean_within_var),\n        \"regime_risk_fraction\": float(between_var / total_var) if total_var > 0 else 0,\n        \"estimation_noise_fraction\": float(mean_within_var / total_var) if total_var > 0 else 0,\n    }\n```\n\n---\n\n## 7. Walk-Forward for Feature Selection\n\n### The Core Problem\n\nFeatures selected in fold k may differ from those selected in fold k+1. This is not a bug — it reflects the non-stationarity of crypto markets. But it creates a practical question: which features to use?\n\n### Three Strategies\n\n```python\nimport logging\nfrom collections import Counter\n\nlogger = logging.getLogger(__name__)\n\n\n@dataclass\nclass FeatureSelectionResult:\n    selected_features: list[str]\n    stability_score: float  # oos_predictions: np.ndarray,  # (n_samples, n_models)\n    y_true: np.ndarray,\n    method: Literal[\"sharpe\", \"mse\", \"constrained_sharpe\"] = \"sharpe\",\n) -> np.ndarray:\n    \"\"\"\n    Optimize ensemble weights to maximize OOS Sharpe ratio.\n    \n    Methods:\n    - sharpe: maximize Sharpe of weighted combination\n    - mse: minimize MSE (simple but ignores risk)\n    - constrained_sharpe: maximize Sharpe with weight constraints\n    \"\"\"\n    n_models = oos_predictions.shape[1]\n    \n    if method == \"mse\":\n        # Analytical solution: ordinary least squares\n        # w = (P^T P)^{-1} P^T y\n        P = oos_predictions\n        weights = np.linalg.lstsq(P, y_true, rcond=None)[0]\n        # Normalize to sum to 1\n        weights = weights / np.sum(np.abs(weights))\n        return weights\n    \n    elif method in (\"sharpe\", \"constrained_sharpe\"):\n        def neg_sharpe(w):\n            combined = oos_predictions @ w\n            returns = combined * y_true\n            mu = np.mean(returns)\n            sigma = np.std(returns) + 1e-10\n            return -mu / sigma\n        \n        constraints = [\n            {\"type\": \"eq\", \"fun\": lambda w: np.sum(w) - 1},  # sum to 1\n        ]\n        \n        if method == \"constrained_sharpe\":\n            # Each weight between 0 and 0.5 (no model dominates)\n            bounds = [(0, 0.5)] * n_models\n        else:\n            bounds = [(-1, 1)] * n_models\n        \n        w0 = np.ones(n_models) / n_models\n        result = minimize(\n            neg_sharpe, w0,\n            method=\"SLSQP\",\n            bounds=bounds,\n            constraints=constraints,\n        )\n        \n        return result.x\n    \n    else:\n        raise ValueError(f\"Unknown method: {method}\")\n```\n\n---\n\n## 11.
